Download

Binary files are available for both of Windows and Mac OS, these can be downloaded from: http://prdownloads.sourceforge.net/minion/minion_0.2.1.zip?download

The source code is available through the use of a Subversion server. Subversion is a version control system for source code. Details of how to use subversion are given at the MINION download site: http://sourceforge.net/svn/?group_id=159145 It should be noted that this source code is subject to more changes, than the binaries, and hence is less stable.

The subversion server can also be viewed through a web browser at: href=http://svn.sourceforge.net/viewcvs.cgi/minion/
