Downloading TAILOR

Download Executable

TAILOR comes together with the latest releases of MINION which is available on Minion's download page on Sourceforge.
You can also download the archive tailor.tar.gz containing stand-alone TAILOR v0.2(March 14, 2008).

Source Code over SVN

The most recent version of TAILOR is available over Minion's public SVN repository with the following instruction set:

svn co https://minion.svn.sourceforge.net/svnroot/minion/trunk/translator translatorDir

which checks out the newest version of the translator in the directory translatorDir. The repository contains the binary as well as the whole source code.
