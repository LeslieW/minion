Overview

TAILOR translates problem models formulated in Essence' to the format of the Constraint Solver MINION. Essence' is a solver-independent modelling language and a subset of the abstract specification language Essence.

TAILOR provides a command-line and a graphical version [picture1]. TAILOR is distributed as a Java .jar file and can easily be executed on every platform. There are different translation-options that enhance the problem model, such as different kinds of common subexpression exploition.

It is the aim of our research to produce effective solver instances automatically. For more information about automatic model generation or Essence, click here.
