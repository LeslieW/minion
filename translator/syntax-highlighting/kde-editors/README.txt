================================================
SYNTAX HIGHLIGHTING FOR KDE EDITORS
================================================
support: andrea@cs.st-and.ac.uk

Jan de Muijnck-Hughes kindly provided .xml files 
that allow syntax highlighting of Essence' and 
Minion files in the kde editors kate and kwrite.
The files must end with either .eprime (for 
Essence' highlighting) or .minion .min or .m
(for Minion highlighting).


Files
-----
eprime.xml		Highlighting for Essence'
minion.xml		Highlighting for Minion


Installation
------------

Copy the files eprime.xml and minion.xml into the 
location

~/.kde/share/apps/katepart/syntax/

If the directories don't exist then create them.
Then restart kate or kwrite.
